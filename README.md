# 📝 ToDo App - Task Management System

A simple and efficient **ToDo List Application** built using **Spring Boot** that helps users manage their daily tasks with ease. This project demonstrates full-stack development with modern Java backend technologies and a responsive frontend.

---

## 🚀 Features

* ➕ Add new tasks
* 📝 Update existing tasks
* ❌ Delete tasks
* 📋 View all tasks in an organized manner
* 📱 Responsive UI using Bootstrap
* ⚡ Fast and efficient backend with Spring Boot

---

## 🛠️ Tech Stack

### 💻 Frontend

* Thymeleaf (Server-side rendering)
* Bootstrap (Responsive UI & styling)

### ⚙️ Backend

* Spring Boot
* Spring Data JPA
* Lombok

### 🗄️ Database

* MySQL

## ⚙️ Setup & Installation

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Ayush-kumar-01/ToDo-App.git
cd ToDo-App
```

### 2️⃣ Configure Database

Update `application.properties`:

```properties
spring.datasource.url=jdbc:mysql://localhost:3306/todo_db
spring.datasource.username=your_username
spring.datasource.password=your_password

spring.jpa.hibernate.ddl-auto=update
```

### 3️⃣ Run the Application

```bash
mvn spring-boot:run
```

### 4️⃣ Open in Browser

```
http://localhost:8080
```
